# Animaciones Implementadas

Este directorio documenta las animaciones que han sido implementadas en el sitio web después de ser prototipadas con HyperFrames.

## Proceso de Animaciones

1. **Prototipado**: Se crea una composición en `hyperframes/compositions/`
2. **Revisión**: Se genera video en `hyperframes/output/` y se valida
3. **Implementación**: Se traduce a código CSS/JS en el proyecto
4. **Documentación**: Se registra aquí la animación implementada
